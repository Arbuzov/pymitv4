pymitv4

A Python 3 module to control Xiaomi TV 4

import TV from pymitv4

tv = TV('192.168.0.32')
tv.turn_on()

Get started

1

Install using pip

To install, open your command line and pop in the following: pip install pymitv4.

2

Start using

To start using, look at the example above or look at the documentation.
